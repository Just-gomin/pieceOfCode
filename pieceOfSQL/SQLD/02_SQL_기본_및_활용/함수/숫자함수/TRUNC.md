# TRUNC

#SQLD #SQL #function

- 입력: 수, \[자리수]
- 출력: 주어진 수를 버림한 결과를 반환
  - 자리수가 주어진 경우: 자리수까지 버림한 결과를 반환한다.
  - 자리수가 안 주어진 경우: 버림한 정수를 반환한다.

```SQL
SELECT TRUNC(-2.3), TRUNC(2.3), TRUNC(5.4378, 2);
```

|     | TRUNC(-2.3) | TRUNC(2.3) | TRUNC(5.4378, 2) |
| --- | ----------- | ---------- | ---------------- |
| 1   | -2          | 2          | 5.43             |
