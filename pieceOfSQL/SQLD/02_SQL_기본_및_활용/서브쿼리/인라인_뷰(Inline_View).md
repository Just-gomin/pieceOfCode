# 인라인 뷰(Inline View)

#SQLD #SQL #subquery

FROM절 등 테이블명이 올 수 있는 위치에 사용 가능하다.

```SQL
SELECT PR.PRODUCT_CODE,
		P.PRODUCT_NAME,
		P.PRICE,
		PR.MEMBER_ID,
		PR.CONTENT
FROM PRODUCT_REVIEW PR
	(SELECT PRODUCT_CODE,
			PRODUCT_NAME,
			PRICE
	FROM PRODUCT ) P
WHERE PR.PRODUCT_CODE = P.PRODUCT_CODE;
```
